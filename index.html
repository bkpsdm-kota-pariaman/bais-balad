Wajib Install Aplikasi

Aplikasi ini hanya dapat digunakan setelah di-install ke Layar Utama (Home Screen) perangkat Anda. Tampilan login hanya akan muncul pada aplikasi yang sudah ter-install.

Panduan Install Manual

PERHATIAN

Anda WAJIB menggunakan Google Chrome (Android) atau Safari (iPhone/iOS). Browser lain tidak didukung!

Android (Google Chrome):

  1. Pastikan Anda sedang membuka link ini di Google Chrome.
  2. Ketuk ikon titik tiga (⋮) di pojok kanan atas browser.
  3. Pilih menu "Add to Home Screen", "Tambahkan ke Layar Utama", atau "Install Aplikasi".
  4. Ikuti petunjuk di layar hingga proses selesai.
  5. Setelah berhasil, tutup Chrome ini lalu buka aplikasi dari ikon baru di Layar Utama HP Anda.

iOS / iPhone (Safari):

  1. Pastikan Anda sedang membuka link ini di Safari.
  2. Ketuk ikon Share di barisan bawah layar.
  3. Geser menu ke bawah, lalu cari opsi "Add to Home Screen" atau "Tambahkan ke Layar Utama" .
  4. Ketuk "Add" atau "Tambah" di sudut kanan atas.
  5. Setelah berhasil, tutup Safari ini lalu buka aplikasi dari ikon baru di Layar Utama iPhone/iPad Anda.
