Tentang BAIS BALAD

BAIS BALAD adalah aplikasi absensi online yang dirancang khusus oleh BKPSDM Kota Pariaman untuk kegiatan ASN di lapangan.

Scan QR Cepat

Absen cepat pakai kamera HP, tinggal scan QR Code kegiatan.

Validasi GPS & Foto

Lokasi dan foto selfie Anda memastikan absensi akurat dan sah.

Performa Terbaik

Dengan teknologi Progessive Web App (PWA), aplikasi bisa di install di perangkat dengan browser safari, google chrome, firefox dan microsoft edge.

Cara Pasang Aplikasi di HP

Android (Chrome)

  1. Buka aplikasi dengan menekan tombol "Buka Aplikasi".
  2. Tekan tombol menu (tiga titik ⋮) di pojok kanan atas.
  3. Pilih menu "Instal aplikasi" atau "Tambahkan ke Layar Utama".
  4. Ikuti petunjuk yang muncul. Aplikasi akan terpasang di layar HP Anda.

iPhone (Safari)

  1. Buka aplikasi dengan browser Safari.
  2. Tekan tombol Bagikan (ikon kotak dengan panah ke atas).
  3. Gulir ke bawah, lalu pilih "Tambahkan ke Layar Utama".
  4. Konfirmasi dengan menekan "Tambah". Ikon akan muncul di layar iPhone Anda.

Firefox

  1. Buka aplikasi dengan menekan tombol "Buka Aplikasi".
  2. Tekan tombol menu (tiga titik ⋮) di pojok kanan bawah/atas.
  3. Pilih menu "Install".
  4. Aplikasi akan terpasang di layar HP Anda.

Bantuan & Tanya Jawab

Kenapa saya tidak bisa login?

Pastikan NIP dan NIK (sebagai password) yang Anda masukkan sudah benar dan terdaftar di data kepegawaian. Jika masih belum bisa, silakan hubungi admin BKPSDM Kota Pariaman untuk bantuan verifikasi data.

Saya menekan tombol install, tapi tidak ada yang muncul. Bagaimana cara install manual?

Beberapa browser mungkin tidak langsung menampilkan pesan install. Jika ini terjadi, Anda bisa menginstall aplikasi secara manual. Silakan lihat panduan "Cara Pasang Aplikasi di HP" di atas untuk langkah-langkah spesifik sesuai browser Anda (Chrome, Safari, atau Firefox).

Aplikasi meminta izin kamera dan lokasi, apakah aman?

Ya, sangat aman. Izin kamera hanya untuk mengambil foto selfie saat absen sebagai bukti kehadiran. Izin lokasi (GPS) hanya untuk memastikan Anda berada di lokasi kegiatan yang telah ditentukan. Data Anda tidak akan digunakan untuk keperluan lain di luar sistem absensi ini.

Aplikasi menampilkan "Pembaruan Tersedia", apa yang harus saya lakukan?

Cukup tekan tombol "Update Sekarang". Aplikasi akan memuat ulang secara otomatis dengan versi terbaru agar Anda selalu mendapatkan fitur dan perbaikan terkini. Proses ini cepat dan tidak akan menghapus data login Anda.
